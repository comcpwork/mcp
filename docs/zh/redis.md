## 使用技巧

1. **数据类型**：Redis 支持字符串、列表、集合、哈希等
2. **过期时间**：使用 TTL 管理会话
3. **模式匹配**：在生产环境谨慎使用通配符
4. **管道操作**：可以一起执行多个命令
